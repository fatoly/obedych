# Глоссарий

**Версия:** 1.0 · Июнь 2026

---

## Проект и продукт

- **БЦ** — бизнес-центр
- **Дайджест** — ежедневный пост канала БЦ со списком доступных бизнес-ланчей
- **Дневник питания** — модуль 4, ведение приёмов пищи с расчётом КБЖУ
- **КБЖУ** — калории, белки, жиры, углеводы (макронутриенты)
- **Mini App** — Telegram WebApp, полноценный интерфейс продукта внутри Telegram
- **Фоллбэк** — вывод последнего известного меню с пометкой о дате, если на сегодня данных нет

## Экономика La Coin

- **La Coin (ЛК, ланч-коин)** — валюта сообщества и расчётная единица экономики; 1 ЛК = 1 обед = 1 бизнес-ланч (см. [`LA_COIN.md`](LA_COIN.md), [`MONETIZATION.md`](MONETIZATION.md))
- **Благотворительный обед** — бесплатный обед, за который повар получает новую монету (эмиссия)
- **Донорский фонд** — публичный резерв монет, пополняемый внешними пожертвованиями
- **Эмиссия / циркуляция** — создание новых монет против оборота уже существующих

## Метрики

- **MAU / DAU** — месячная / дневная активная аудитория
- **Retention D30** — доля пользователей, вернувшихся в продукт через 30 дней после регистрации
- **MRR** — Monthly Recurring Revenue, ежемесячная регулярная выручка
- **LTV / CAC / ARPU** — пожизненная ценность клиента / стоимость привлечения / средняя выручка на пользователя
- **Dogfooding** — практика, когда команда сама пользуется своим продуктом

## Бизнес и право

- **MVP** — Minimum Viable Product, минимально жизнеспособный продукт
- **GTM** — Go-to-market, стратегия выхода на рынок
- **B2B / B2C / C2C** — компания-компании / компания-потребителю / потребитель-потребителю
- **152-ФЗ** — закон о персональных данных
- **422-ФЗ** — закон о самозанятых
- **НПД** — налог на профессиональный доход (режим самозанятых)
- **ERID** — единый идентификатор рекламы в ЕРИР
- **ЕРИР** — Единый реестр интернет-рекламы (Роскомнадзор)
- **ОРД** — оператор рекламных данных
