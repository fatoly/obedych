## Продуктовые и операционные риски

| Риск | Вероятность | Митигация |
|---|---|---|
| Заведения не обновляют меню | Высокая | Гибрид парсинга + верификации; фоллбэк на вчерашнее меню |
| Низкая дисциплина самозаполнения | Высокая | Парсинг основной, бот администратора — дополнительный |
| Малая аудитория канала на старте | Средняя | Договорённость с УК БЦ о продвижении; компенсация команды создаёт демо-эффект |
| Яндекс / 2ГИС блокируют парсинг | Средняя | Диверсификация источников, прямые партнёрства с заведениями |
| Дневник питания не используется | Средняя | Одна кнопка для ланча, напоминания, геймификация |
| Накрутка аудитов ради премиума | Высокая | Фото чека с геометкой + фото блюда; иначе не засчитан |
| Низкое качество аудит-отчётов | Средняя | Жёсткая форма; оценка по рубрике; рейтинг аудиторов |
| Экономика аудита не сходится для пользователя | Высокая | Компенсация чека закладывается в пакет заведения |
| Риск признания постов аудиторов рекламой | Средняя | Посты добровольны и неоплачиваемы; монетизируется производство; Instagram исключён |
| Нет прав на перепродажу купленного контента | Высокая | Передача прав в оферте с аудитором |
| Изображение третьих лиц в кадре (ст. 152.1 ГК) | Средняя | ТЗ запрещает съёмку лиц |
| Себестоимость пост-продакшна не масштабируется | Средняя | AI-конвейер (авто-нарезка, авто-субтитры, цветокор) |
| Отравление едой Домашнего повара | Средняя | Повышенная проверка повара; пилот в доверенном коммьюнити |
| Приватность данных питания перед HR | Средняя | Только обезличенная агрегированная аналитика |

## Риски экономики La Coin

Касаются внутренней валюты сообщества (см. [`LA_COIN.md`](LA_COIN.md)).

| Риск | Вероятность | Митигация |
|---|---|---|
| Инфляция: монет больше, чем реальных обедов | Высокая | Эмиссия только из обеспеченных источников (благотворительный обед, покупка, донорский фонд); приветственный бонус конечен |
| Фиктивные транзакции ради монет | Высокая | Обязательное подтверждение факта получения еды едоком; закрытое сообщество с репутацией |
| Заморозка ликвидности: монеты оседают, новым не на что начать | Средняя | Донорский фонд и покупки как постоянный приток; приветственный бонус новым участникам |
| Дефицит поваров (готовить тяжелее, чем есть) | Средняя | Благотворительный обед даёт дополнительный стимул готовить; чередование ролей |
| Юридическая квалификация La Coin | Средняя | Позиционирование как не-платёжного средства; проверка с юристом (см. [`LEGAL.md`](LEGAL.md)) |
