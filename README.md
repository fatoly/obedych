# Обедыч

Персональный агрегатор бизнес-ланчей для офисных сотрудников в Telegram.

Сервис ежедневно собирает меню кафе и ресторанов вокруг конкретного бизнес-центра и присылает сотруднику персональный дайджест с учётом его вкусов, бюджета и пищевых ограничений. Дополнительно ведёт дневник питания (КБЖУ) и служит платформой для аудита заведений и корпоративных wellbeing-программ.

**Домен:** obedych.ru · **Канал:** Telegram · **Город:** Новосибирск  
**Статус:** MVP — разработка не начата  
**Инициатор:** Клуб технологического предпринимательства Школы 21

---

## Как это работает

Три точки касания, одна пользовательская база:

1. **Канал БЦ** — в 11:00 каждый будний день публикует список бизнес-ланчей в радиусе пешей доступности от бизнес-центра
2. **Персональный бот + Mini App** — рекомендация дня, дневник КБЖУ, история обедов, фильтры по кухне и цене
3. **Городской канал** — медиа об общепите города: открытия, акции, подборки

## BL Coin

Внутренняя валюта экосистемы — **BL Coin (бизнес-ланч коин)**. Один BL Coin равен одному бизнес-ланчу. Монета обеспечена реальной едой: участник кампуса приготовил ланч и принёс в контейнере — заработал BL Coin; другой участник получил обед — потратил BL Coin. Рубли внутри сообщества не нужны.

Роли повара и едока свободно чередуются: сегодня ты принёс контейнер и заработал монеты, завтра потратил их на ланч, который принёс кто-то другой. Монеты накапливаются и тратятся в удобный момент — у кампусных поваров, в ресторанах-партнёрах или на подписку в боте.

## Команда

Проект строится студентами Школы 21 по веткам обучения:

- **Project Management** → Project Manager
- **Product Management** → Product Manager
- **Бизнес и системный анализ** → Бизнес-системный аналитик
- **Разработка** → Backend, Telegram Bot + Mini App, Инженер данных
- **Тестирование** → QA-инженер, Специалист по ИБ
- **Dev-ops** → DevOps-инженер
- **Дизайн** → UX/UI-дизайнер

Каждый участник получает **1 БЛ в неделю** — сертификат на бизнес-ланч в партнёрском заведении. Компенсация одновременно является обязательным dogfooding-правилом: все члены команды пользуются продуктом.
